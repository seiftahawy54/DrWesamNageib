# Dr Wesam Nageib's Official website

This is Dr. Wesam Nageib official website code repo.

NodeJS:     Backend \
Postgres:   Database \
Sequelize:  ORM \
EJS:        Template Engine \
Jest:       Testing \
Sass:       CSS Preprocessor

### Dependencies installation

`` yarn install ``

### Starting application

`` yarn start ``

### Run Test (under_construction)

`` yarn Test ``
